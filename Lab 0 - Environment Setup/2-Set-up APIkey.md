# Creating a Cloud API Key
1. Navigate to [IBM Cloud](cloud.ibm.com). Click on **Manage** -> **Access(IAM)**

   <img width="784" height="343" alt="image" src="https://github.com/user-attachments/assets/312e511a-3f67-479e-868c-535488d10a2e" />

2. On next screen, select **API Keys** from menu.

   <img width="783" height="431" alt="image" src="https://github.com/user-attachments/assets/bb8622cb-dfb9-4059-aa75-391d5275887f" />

3. Click on **Create**.

   <img width="782" height="433" alt="image" src="https://github.com/user-attachments/assets/a5b76a22-5dd0-46c0-93da-9e5fdc3c0e23" />


4. Give your API key a name, then click on **Create**.

   <img width="783" height="433" alt="image" src="https://github.com/user-attachments/assets/9f084792-a960-41ee-b91a-2f6de5524034" />

5. Copy your API key and save it in a secure location. You will need it in later steps.
